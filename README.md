A word-ladder puzzle solver.  You can replace the dict.txt file with your own dictionary of
words.

Examples:

    $ ./ladder.py todd john
        TODD
        TOAD
        LOAD
        LOAN
        JOAN
        JOHN

    $ ./ladder.py geek wire
        GEEK
        WEEK
        WEED
        WELD
        WILD
        WILE
        WIRE
